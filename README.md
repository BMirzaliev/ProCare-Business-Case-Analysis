# ProCare-Business-Case-Analysis
This repository contains a group project analyzing the ProCare Solutions business case using financial data and Power BI visualization.

## Project Overview
The analysis focuses on three healthcare projects:
- Cleveland Clinic
- Sacred Heart Hospital
- UCLA Medical Center

The project evaluates key financial indicators such as:
- Revenue trends
- Cost evolution
- Profit margins
- Cost margins
- Patient and revenue correlation

## Key Insights
- **UCLA Medical Center** achieved the highest overall profit margin (~43%).
- **Sacred Heart Hospital** showed declining profitability due to rapidly increasing costs.
- **Cleveland Clinic** demonstrated stable growth with improving margins.
- **External service providers** represent the largest cost component across all projects.

## Tools Used
- Excel (data preparation)
- Power BI (data visualization and dashboard)
- Financial data analysis
